ABOUT THE WEBSITE

The Boys is a series live on Amazon Prime. Superheroes are often as popular as celebrities, as influential as politicians, and sometimes even as revered as gods. But that's when they're using their powers for good. The Series has a total of 3 seasons with 8 episodes each and a fourth season is on its way. This website has information for some of the main characters of the show. There are two main APIs used for the website. Website animation API is for the users a more apealing look. Picture in picture API is for the user to be able to browse around the website without having to close the video their watching.
